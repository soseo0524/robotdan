# robotdan

## 1. 전체 시스템 구성 및 역할
시스템은 크게 자율주행 로봇(Pinky)과 로봇팔(JetCobo)로 이루어진다 

### 자율주행 자동차 (Pinky 1, 2, 3):
Pinky 1, 2: 원자재 및 부품 운반을 담당하며, 조립 공정에 필요한 자재를 적시에 공급
Pinky 3: 완성된 제품 중 합격품만 포장 및 출하 구역으로 이송

### 로봇팔 (JetCobot A, B):
로봇팔 1 (A): 자재 조립 요청에 따라 물품을 조립하고 운송차(Pinky)에 상차
로봇팔 2 (B): 완성품의 외관 및 기능을 검사하여 합격/불합격(OK/NG)을 분류

### 중앙 제어 시스템: 
전체 공정을 모니터링하고 관리하며, 각 위치별 적재 수량 및 불량률 등을 데이터베이스(DB)




## 2. 수정된 물류 공정 시나리오 (데이터 관점)
### [단계 1] 조립 및 중간 선반 적재 (JetCobot A)
동작: 자재를 조립한 후, 로봇팔 2와의 사이 공간인 '중간 선반(Buffer Shelf)'에 올려놓는다.

DB 기록:
Inventory 테이블에서 해당 물품의 위치를 'ASSEMBLY_ZONE' -> 'BUFFER_SHELF'로 update 
제품의 상태는 'ASSEMBLED'로 변경 

### [단계 2] 품질 검수 및 상차 (JetCobot B)
동작: 선반 위에 놓인 물품을 집어 카메라로 AI 검수

DB 기록:
검사 결과에 따라 status를 'OK' 또는 'NG'로 결정 

  -> 'OK' 판정 시: 해당 물품을 대기 중인 Pinky 3에 상차하고, location을 'PINKY_3'로 변경.
  
  -> NG' 판정 시: 불합격 존으로 폐기하며 기록
